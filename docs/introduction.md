# Introduction

Ved is a deterministic control-plane programming language designed for
building reliable long-running distributed systems.

Modern software systems consist of many interacting services, workflows,
and autonomous components. The logic that governs how these systems scale,
recover, and coordinate is often implemented using ad-hoc scripts or
framework-specific controllers.

Ved explores a programming model where system behaviour is expressed through:

- explicit persistent state
- deterministic execution
- goal-driven convergence
- structured authority boundaries
- replayable side-effect handling

Ved programs are not intended to implement application business logic.
Instead, they describe how complex software environments should evolve
towards stable operating conditions.

The project is currently in early design and prototyping stages.
