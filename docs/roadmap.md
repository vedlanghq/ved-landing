# Roadmap

Ved development is staged to validate core ideas incrementally.

## Phase 1 — Deterministic Runtime Core

- bytecode execution loop
- append-only state journal
- snapshot recovery
- basic goal evaluation

## Phase 2 — Language Frontend

- syntax stabilization
- AST and IR design
- semantic validation pipeline

## Phase 3 — Operational Tooling

- replay inspection tools
- convergence visualization
- structured observability

## Phase 4 — Distributed Runtime Research

- multi-node scheduling
- replicated persistence
- domain placement strategies

Timelines are exploratory and subject to change as architectural insights
emerge.
